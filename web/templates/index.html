About This Tool

How It Works

  1. Load Light Curve: Read Kepler FITS data (TIME, PDCSAP_FLUX)
  2. Detrend: Apply median filter to remove stellar variations
  3. Detect Dips: Find peaks in inverted flux (transits)
  4. Estimate Period: Median spacing of transit times
  5. Assess Stability: Coefficient of variation of period
  6. Score Candidates: Multi-factor ranking (quality, period, stability)
  7. Triage Results: Categorize by scientific confidence

Output Files

  • transit_results.csv: All candidates ranked by score
  • top_candidates.csv: "review_now" priority cases
  • caution_candidates.csv: "review_with_caution" cases

Scoring Rubric

Base (0-55 pts): Quality flag (high/medium/low confidence)

Period (0-35 pts): +25 if detected, +10 if 0.5-50 days

Stability (±8 pts): +8 stable, +3 moderate, -8 unstable

Peak Count (±20 pts): Bonus/penalty based on peak range

Data Source

Kepler mission long-cadence (LLC) light curves from:

https://archive.stsci.edu/pub/kepler/lightcurves/

Files are ~10-15 MB each (30min-1hr processing time for 10 targets)
